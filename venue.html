Venues

Overview

This section covers the creation and editing of venues. It also covers the creation of clinics which is the combination of a venue, a clinician and a date which is refered to as a clinic day.

The areas we will cover are:

  1. The venues list, which enables the selection or creation of a venue.
  2. The venue details, which enables you
  3. The clinic days, which enables you create/edit a clinic.

How to..

Add a Venue

  1. In the Venue List view click the Add button
  2. [Required] Enter the Venue Name.
  3. Click colour dropdown and choose a colour for the venue.
  4. The switch defines whether the venue is active or not (.i.e., It available for use in the system - if inactive it won't be visible to use except in the venue list)
  5. Enter an address for the venue.
  6. Click on the Hours tab to switch to the default hours for venue. This pre-populates the clinic day entries.
  7. Edit this if required.
  8. Click on the Directory tab to switch to the default directort for venue.
  9. Edit this if required.
  10. Click on the Documents tab to switch to the default documents for venue. The documents here can be used to attach to emails.
  11. Edit this if required.
  12. Click on the SAVE button when you are ready.

Edit a Venue

  1. In the Venue List view click the Edit button at the bottom of the desired venue card.
  2. See above for fields information.
  3. Click on the SAVE button when you are ready.

Add a Clinic Day

  1. In the Venue List view click the Days button at the bottom of the desired venue card.
  2. Select the clinician who will run the clinic.
  3. Click the Add button. A line will appear with the cursor at the date entry.
  4. Enter a date. When entered the system will fill in Start Time and end time from the defualts set in the venue page.
  5. Edit the start and end times if you wish.
  6. If you wish to add any breaks, then click the under the breaks heading. You will be presented with a start and end time for the break. Once you have filled them in then click the .
  7. Once you are happy with the date then click the at the end of the line.
