Overview

This section covers the adding and editing of clients, their cases and manage communication with them.

The two initial sections are:

  1. The clients list, which enables the selection or creation of a client.
  2. The client details, which enables you access/create a case, edit details and various other options.

List View

The page shows the list of clients in the system. It can be filtered and sorted by using the column headers (See below). Clicking on any of the clients will take you to their client page.

The page is split into five areas:

  1. ToolBar
  2. Filter Bar
  3. Client Table Header
  4. Client List
  5. The Navigation Bar
  6. Toolbar

Toolbar

The toolbar has three buttons:

Icon Label Position Purpose
Export Left This exports the client list to a pdf.
Add Wizard Right This opens the add client wizard.
Add Right This opens the add client page.

Filter Bar

This is a set predefined filters

Icon Label Position Purpose
Waiting Left Filters the clients to show those on the waiting list.
Right This text field will filter the clients by name and referance.

Table Header

This contains the header information for the table. It has two features:

Sort If you click on any of the headers it will sort to that field. If you click it again it will reverse the sort and clicking it again will remove the sort.
Filter Each field is filterable by clicking on the ‘filter’ icon and a small pop-up will open giving you options to consider.

Table List

This is the list of available clients. If you wish to open a client click the relevant line.

Navigation Bar

This show the numbers of records on the right and navigation buttons on the left.

Cases

What are cases?

Cases are for each issue that a client presents with. A case contains all the related documents, appointments and process notes.

Case Timelines

This is utility for clinician to map the timeline of a client. It includes the date of birth (Created from the client) and the date of the incident (Created from the Case). The rest of the entries are created by the clinician or administrator.

When seen through the case the initial view of the timeline you will see interconnected time moments.

You can click Add to add a new timeline entry, or under each entry there is two buttons. The first removes an entry and the other allow you to edit an entry.

  • How to Add/Edit a timeline entry.

How to..

Add a Client

  1. In the Client List view click the Add button
  2. Enter the clients name:
    • Enter the title field.
    • Enter the forename
    • [Required] Enter the last name.
      • The title field will set the gender in some cases. e.g. if Mr is selected then the gender will be set to Male.
  3. Enter the gender.
  4. Enter the Date of Birth.
    5. The date of birth will be added to the timeline automatically on saving.
  5. [Required] Select the Registered Clinic from the drop down. This would be normally be the clinic closest to the client address.
    6. If you set the postcode field and press return the closest clinic will be suggested.
  6. [Required] Select the clinician from the drop down. This can de dictated by the letter of instruction or the therapy clients preference.
  7. [Optional] Putting the client on the waiting List:
    • click the on waiting list check box.
    • Select a Waiting List Reason for the client being put on the waiting list.
  8. Click on the Save button

Add a Client with the Client Wizard

This is the move convenient method to add a new client. It covers the creation of the client, the case, associated documents and the initial appointment.

The wizard steps you through several sections. In each section, you must fill in all required fields before being able to progress to the next. You can navigate backwards and forwards through the completed sections by using the PREVIOUS and NEXT buttons if you wish to correct or review what you have entered before saving.

Client Information will be used to populate the report so extra care must be taken here.
  1. In the Client List view click the Add Wizard button

Section 1: Client Details

Enter the Client Information

  1. Enter the clients name:
    • Enter the title field.
    • Enter the forename
    • [Required] Enter the last name.
      • The title field will set the gender in some cases. e.g. if Mr is selected then the gender will be set to Male.
  2. Enter the gender.
  3. Enter the Date of Birth.
    4. The date of birth will be added to the timeline automatically on saving.
  4. [Required] Select the Registered Clinic from the drop down. This would be normally be the clinic closest to the client address.
    5. If you set the postcode field and press return the closest clinic will be suggested.
  5. [Required] Select the clinician from the drop down. This can de dictated by the letter of instruction or the therapy clients preference.
  6. [Optional] Putting the client on the waiting List:
    • click the on waiting list check box.
    • Select a Waiting List Reason for the client being put on the waiting list.
  7. Click on the NEXT button to go to the next section (Contact Info.)

Section 2: Contact Info

This section is for you to add client contact details and who is paying

You may need to scroll up to see the top of this page depending on the width of the browser window.
  1. Select if the Instructing party is a Client (individual) or Contact (company). This will determine what drop down appears in the next field
  2. Select Instructing party from the drop-down list.
  3. Select if the Solicitor is a Client (individual) or Contact (company)
  4. Select the Solicitor from the drop-down list.
    If the instructing party or Solicitor is not in the list select Add contact to bring up a dialog to enter a new contacts details.
  5. [Required] Select the Payer (this is usually the instructing party) from drop down (Client, Relative, Instructing Party or Solicitor)
  6. If contact information is available then click on ADD to enter details for Email, mobile, landline, etc
    • Select the type email,mobile,etc.
    • Select if it is a home or work contact
    • Enter the number/address in the data section
    • Click on the tick button to add the contact info to the directory.
    • Repeat if additional items are available
  7. Click on NEXT this takes you to Case setup

Section 3: Case

This section is to add the case information.

You may need to scroll up to see the top of this page depending on the width of the browser window.
  1. [Required] Select the type of case from the drop down (RTC, Work, Clinical Negligence etc)
  2. Enter the Payer Reference
  3. Enter the Solicitor’s Reference
  4. Set On Hold flag if required
  5. Enter an initial comment
  6. Enter the Incident date
  7. Click on the NEXT > button to go to the next section (Documents)

Section 4: Documents

This section is to add any documents associated with the case/client.

  1. If documents are available then click on ADD to enter details and/or upload document
  2. Click on Select files… to upload the Case files previously saved into Process/Client Notes/clinic/clinic date/client
  3. Navigate to the files and click on Open to add the files to inClinic
  4. Enter details into the document Manifest. This is used to:
    • keep a track of what documents exist for the case
    • to help us identify if the number of document pages require additional fees.
  5. Enter the title e.g. filename
  6. Enter the Source date – this is the date on the document
  7. Enter the Author – this is only necessary for expert report
  8. Select the type of document (LOI, GP Notes, Expert report etc)
  9. No of pages – this is required for records and reports
  10. Receipt date (defaults to today)
  11. Click on the tick to save the manifest record
  12. Click on the NEXT button to go to the next section (Initial Appointment)

Section 5: Initial Appointment

    If the client is on the waiting list then you can jump to the save step.
  1. If the client has already confirmed that they will attend the appointment tick the Appointment Confirmed check box
  2. Navigate to the date (as previously identified) and select from the shown available times
  3. Select a time slot from the drop down
    4. All other fields should be filled in and the charges should automatically be filled in.
  4. Click on the Save to complete the wizard.
    5. You will be prompted to add a further client or view list – select as required When the wizard is complete the date of birth and date of incident are used to create entries in the clients case timeline.

Add a Client to the Waiting List

  1. In the Client List view select a client
  2. Select Details from the sub-menu
  3. check the waiting list check box.
  4. [Required] Select a Waiting List Reason for the client being put on the waiting list.
  5. Click on the Save button
    6. If the client is on the waiting list it is visible in the client list (see on waiting List header) and client card (under the sub menu)

Remove a Client from the Waiting List

  1. In the Client List view select a client
  2. Select Details from the sub-menu
  3. uncheck the waiting list check box.
  4. Click on the Save button
    5. The client is automatically removed from the the waiting list when an appointment is created for them. If you cancel this appointment you will have to manually put them back on the waiting list (see Above)

How to Add/Edit a case timeline entry

  1. To add or edit an entry then you can click '+ Add' to add a new timeline entry or the pen icon allow you to edit an entry
  2. [Required] The first field is the type and the options are Event, Entry or Extract. The 'Event' type is to add significant milestones. The 'Entry' type is for general entries and finally, the 'Extract' is for identifying extracts that are from third-party documents. If the extract is assigned then you have further options to identify the source of the document.
  3. [Required] The date of the entry/Event.
  4. [Required] A header for the entry to make easily identifiable.
  5. The next field is the body text of the event where a more detailed description can be created. It is a full WYSIWYG editor. It has a special feature called OCR. The OCR button will allow you to upload a scan or image of text and convert it into an editable text.
  6. If 'Extract' Type is chosen then you have additional fields: ​​​​​​​
      ​​​​​​​
    1. ​​​​​​​ The Manifest Entry drop-down enables you to select from the recorded manifest of documents. It will automatically fill the remaining fields if it exists.​​​​​​​​​​​​​​ ​​​​​​​
    2. The Title field is for the document title
    3. The Author field is for the author of the document
    4. Written on is for date of the document creation
